Repository Setup

Finny Website

finny-website (Github)

  1. Clone the repository

  2. Run bin/setup

  3. After the setup script succeeds, run bin/start to start the development server.

  4. Visit http://www.finny.dev:8080 in your browser

Finny API

finny-api (Github)

  1. Clone the repository

  2. Run bin/setup

  3. After the setup script succeeds, run bin/start to start the development server.

  4. Run curl http://api.finny.dev:8080/status to check that the developer server is running.
