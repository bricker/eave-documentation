This document provides an overview of the Credit Application System at Finny.

Website

finny-website (Github)

A form built with React, which accepts the following inputs:

  1. Applicant Name

  2. Applicant Social Security Number

  3. Applicant Email

  4. Applicant Address

  5. Applicant Annual Income

  6. Applicant Employment History

When this form is submitted, the data is sent to api.finny.com

API

finny-api (Github)

The API endpoint POST /application performs the following steps:

  1. Reads and parses the user input from the JSON-formatted request body

  2. Validates the data

  3. Saves the data to the database

Credit Application Processor

finny-credit-application-processor (Github)

Periodically scans the database for pending applications and processes them. To process an application, the code performs the following steps:

  1. Pulls the applicant's FICO credit score from Experian

  2. Verifies the applicant's employment history.

  3. Compares the FICO credit score to the configured threshold value.

    • If the FICO credit score is below 720, sends an email to the applicant denying the application.

    • If the FICO credit score is equal to or above 720, sends an email to the applicant approving the application.
